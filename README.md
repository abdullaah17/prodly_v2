# PRODLY v2 - Role-Based Onboarding Engine

A complete desktop SaaS-style application demonstrating multi-language system integration, advanced data structures & algorithms, and professional UI design.

## Quick Start

1. **Build C++ Core:**
   ```bash
   cd Prodly/cpp_core
   g++ -std=c++11 -I./include src/*.cpp main.cpp -o prodly_core.exe
   ```

2. **Run Java GUI:**
   ```bash
   cd Prodly/java_gui
   # Using NetBeans: Open project and run
   # Or compile and run manually
   ```

3. **Login:**
   - Username: `admin`
   - Password: `admin123`
   - Or create a new account

## Architecture

- **C++ Core**: All business logic, DSAs, engines
- **Java GUI**: Professional desktop UI, validation, visualization
- **Integration**: File-based IPC (JSON)

## Key Features

✅ Role-based authentication (Employee, Manager, Admin)  
✅ Personalized onboarding paths  
✅ Task tracking with dependencies  
✅ Progress analytics  
✅ Smart recommendations  
✅ Professional desktop UI
